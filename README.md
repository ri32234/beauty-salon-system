# Beauty Salon Appointment and Management System

## Project Description
The Beauty Salon Appointment and Management System is a software engineering project designed to help beauty salons manage appointments, customer records, employees, and salon services efficiently.

The system provides a digital solution for online appointment booking and salon management.

---

## Features

### Customer Features
- User registration and login
- Online appointment booking
- Appointment cancellation
- View salon services
- Appointment history

### Receptionist Features
- Manage appointments
- View customer records
- Handle booking requests

### Administrator Features
- Manage salon services
- Manage employees
- Monitor appointments
